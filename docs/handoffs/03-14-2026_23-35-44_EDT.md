## Technical Notes
- WMI requires `pythoncom.CoInitialize()` in threaded contexts — must be called at start of any thread using `wmi.WMI()`
- Gemini's `responseMimeType: "application/json"` is only set for first analysis (structured), not for follow-up chat (freeform text)
- Chat follow-ups include hardware summary in system prompt so AI always knows the user's specs
- `_describe_image()` tries Gemini first (free), falls back to Claude — returns text description that gets prepended to Ollama's prompt
- Server heartbeat uses `SERVER_START_TIME` (Unix timestamp set at module load) — frontend polls `/api/status` every 3s, reloads if timestamp changes
