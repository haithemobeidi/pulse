## Technical Notes
- Ollama runs on Windows, reachable from WSL at `http://172.21.0.1:11434` (gateway IP). Set `OLLAMA_HOST=0.0.0.0` in Windows env vars.
- EDID monitor detection uses `Get-CimInstance -Namespace root\wmi -ClassName WmiMonitorID` - far more reliable than `Get-PnpDevice -Class Monitor`
- GPU driver version conversion: WMI gives `32.0.15.9576`, last two parts concatenated (`159576`), take last 5 digits (`59576`), insert dot after 3rd (`595.76`)
- User has RTX 5090 (32GB VRAM), AW3425DW (Alienware) + LG ULTRAGEAR monitors, 16-core CPU, 93.6GB RAM
