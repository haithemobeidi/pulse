## Technical Notes
- Confidence uses Laplace smoothing: `(successes + 1) / (total + 2)` — avoids 0/0 and extreme values from small samples
- Fact deduplication uses Jaccard similarity on word overlap (>0.4 threshold) — Phase 2D will upgrade to vector similarity
- Brain context assembly filters out facts below 0.15 combined score (confidence * decay * relevance)
- Activation tiers: Hot (accessed within 7d), Warm (30d), Cool (30d+) — nightly decay cron updates these
- Outcome prompt uses callback pattern: `promptForOutcome(sessionId, () => resetChat())` — waits for click before resetting
